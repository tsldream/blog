# 相对路径与绝对路径

## 绝对路径

- 省略协议
- 省略协议，域名，端口

例子：http://a.com/a/b/c/s/a
    /k    --->  http://a.com/k        

## 相对路径

例子：http://a.com/a/b/c/s/a        
     ./k    --->  http://a.com/a/b/c/s/k     ../k   --->http://a.com/a/b/c/k
